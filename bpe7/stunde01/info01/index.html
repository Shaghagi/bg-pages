Manipulation der Elemente eines Arrays

Informationsmaterial · Fach: Informatik

Was sind Arrays?

Arrays sind spezielle Datenstrukturen, die nicht nur einen Wert, sondern eine Vielzahl von Werten speichern können. Man kann sich ein Array wie einen Schrank mit Schubladen vorstellen. Jede Schublade bietet Platz für einen Wert (z. B. Vornamen) und trägt eine Nummer, über die man den Inhalt gezielt wiederfindet.

Wichtig:

  • Die Nummerierung der Schubladen heißt in der Fachsprache Index. Die Speicherplätze sind also indiziert.
  • Die Nummerierung beginnt bei 0. Die erste Schublade hat den Index 0, die zweite 1, die sechste 5.

Beispiel: „Schrank“ für Vornamen

Index 0
Uli
Index 1
Anna
Index 2
Emil
Index 3
Sophia
Index 4
Luise
Index 5
Hans

Deklaration & Initialisierung in Python

In Python werden Arrays mit Listen umgesetzt. Deklaration bedeutet, einen Bezeichner anzulegen (z. B. namensliste). Initialisierung bedeutet, dieser Variablen direkt Startwerte zuzuweisen.

Wann getrennt, wann zusammen? Getrennt (erst deklarieren, dann füllen) eignet sich, wenn die Werte erst zur Laufzeit bekannt werden oder schrittweise entstehen. Zusammen (Deklaration und Initialisierung) ist effizient, wenn die Startwerte bereits feststehen.

append() ist eine eingebaute Listen‑Methode. Aufruf: liste.append(wert). Sie hängt den wert am Ende an und vergrößert die Liste um 1 Element.

Möglichkeit 1: Deklaration und Initialisierung

# Deklaration und Initialisierung
namensliste = ["Uli", "Anna", "Emil", "Sophia"]

#index:        0      1       2       3 
# Ergebnis: ["Uli", "Anna", "Emil", "Sophia"]

Möglichkeit 2: erst Deklaration (leeres Array), danach (im Laufe des Programms) mit Werten füllen

# Deklaration (leeres Array)
namensliste = []

# Zuweisung mit append()
namensliste.append("Uli")
namensliste.append("Anna")
namensliste.append("Emil")
namensliste.append("Sophia")

#index:        0      1       2       3 
# Ergebnis: ["Uli", "Anna", "Emil", "Sophia"]

Hinzufügen und Verändern von Elementen

Python-Listen sind nicht abschließend: Mit append() kannst du im weiteren Programmablauf Elemente am Ende hinzufügen. Append() zählt den Index automatisch +1 nach oben!
Vorhandene Werte können per Zuweisung, z. B. namensliste[2] = "Lorenz" überschrieben werden (Beispiel 3). 

namensliste = ["Uli", "Anna", "Emil", "Sophia"]

# Verändern des ersten Wertes
#index:                                0        1       2         3
namensliste[0] = "Ulli"           # ["Ulli", "Anna", "Emil", "Sophia"]

# Hinzufügen am Ende
#index:                                0        1       2         3        4
namensliste.append("Mara")         # ["Ulli", "Anna", "Emil", "Sophia", "Mara"]

# Beispiel 3: Verändern des dritten Wertes
#index:                                  0        1       2         3        4
namensliste[2] = "Lorenz"          # ["Ulli", "Anna", "Lorenz", "Sophia", "Mara"]

Zugriff auf Array-Werte mit dem Index

Auf einzelne Elemente greift man mit dem [Index] zu, z.B. array[3] liefert das letzte Element am Index 3 (hier: Sophia).
Achtung! Das erste Element hat den Index 0, das letzte den Index n − 1, hier z. B. (4  -  1 = 3).
Im unteren Beispiel ist n = 4, weil wir Menschen bei 1 das zählen anfangen und nicht wie die Maschine bei 0.

#index:          0      1       2       3 
namensliste = ["Uli", "Anna", "Emil", "Sophia"]

print(namensliste[3])  # Ausgabe: Sophia
print(namensliste[0])  # Ausgabe: Uli

Anzahl der Elemente (Länge) ermitteln

Die Anzahl der Elemente eines Arrays/ einer Liste erhält man mit len(array).
Achtung! len(array) beginnt bei 1 das Zählen - so wie wir Menschen! 

#index:          0      1       2       3 
namensliste = ["Uli", "Anna", "Emil", "Sophia"]

anzahl = len(namensliste)
print("Anzahl Namen:", anzahl)   # Ausgabe: Anzahl Namen: 4
Hinweis: Statt „Anzahl der Elemente eines Arrays“ wird oft die Formulierung „Länge des Arrays“ verwendet.

Zugriff auf alle Elemente mit einer for-Schleife – zum Lesen und Schreiben

Beispiel 1: Lesender Zugriff (Werte nur ausgeben)

Die Schleife liest nacheinander jedes Element und gibt es aus. Die Liste selbst wird dabei nicht verändert.

#index:          0      1       2       3 
namensliste = ["Uli", "Anna", "Emil", "Sophia"]

# Hinweis: Schleifenvariable i nimmt folgende Werte an: 0 , 1, 2, 3. 
for i in range(len(namensliste)): 
print(namensliste[i])

# Ausgabe:
# Uli
# Anna
# Emil
# Sophia

Beispiel 2: Schreibender Zugriff (Werte verändern)

Die Schleife nutzt die Indizes, um die Elemente gezielt zu überschreiben. Unten wird an jeden Index der Name "Anton" geschrieben.

#index:          0      1       2       3 
namensliste = ["Uli", "Anna", "Emil", "Sophia"]

# Hinweis: Schleifenvariable i nimmt folgende Werte an: 0 , 1, 2, 3. 
for i in range(len(namensliste)):
namensliste[i] = "Anton"


print(namensliste)
# Ergebnis: ["Anton", "Anton", "Anton", "Anton"]

Vorteil: Auch bei 10 000 Elementen bleibt das Programm gleich kurz – die Schleife übernimmt die vollständige Verarbeitung aller Elemente.
