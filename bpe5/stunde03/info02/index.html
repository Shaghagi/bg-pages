For-Schleife in Python

Von der konzeptionellen Ebene zur praktischen Umsetzung

Einführung

Nachdem wir die konzeptionelle Ebene der For-Schleife besprochen haben, wollen wir uns nun mit der praktischen Umsetzung in Python beschäftigen. Die meisten Probleme treten beim Verständnis des Schleifenkopfs auf, insbesondere bei der Übersetzung von Konzepten wie "Wiederhole von i = 0 solange i kleiner gleich 2" in Python-Syntax.

Wichtig: In Python wird die For-Schleife anders implementiert als in vielen anderen Programmiersprachen. Statt einer Bedingung verwenden wir die range()-Funktion, die eine Zahlenfolge erzeugt.

Der Schleifenkopf in Python

Konzeptionelle Ebene

"Wiederhole von i = 0 solange i kleiner gleich 2"

Dies entspricht:

  • Startwert: i = 0
  • Bedingung: i ≤ 2
  • Schrittweite: 1 (Standardfall: ist immer voreingestellt)

Python-Umsetzung

# Unbedingt den Hinweis lesen!!! # range(3) erzeugt die Zahlenfolge: 0,1,2 for i in range(3): # Anweisungen

Hinweis: Die range()-Funktion in Python erzeugt eine Zahlenfolge von Start bis Ende - 1. Daher müssen wir bei der Bedingung "i ≤ 2" als Endwert 3 angeben. Dann erzeugt range() genau die Zahlenfolge: 0,1,2

Die range()-Funktion verstehen

Die range()-Funktion erzeugt eine Sequenz von Zahlen, über die die For-Schleife iteriert. Dies ist der Schlüssel zum Verständnis von For-Schleifen in Python.

Visualisierung: range(5)

Varianten der range()-Funktion

range(stop)

Erzeugt Zahlen von 0 bis stop-1

for i in range(5): print(i) # Ausgabe: 0, 1, 2, 3, 4

range(start, stop)

Erzeugt Zahlen von start bis stop-1

for i in range(2, 6): print(i) # Ausgabe: 2, 3, 4, 5

range(start, stop, step)

Erzeugt Zahlen von start bis stop-1 mit Schrittweite step

for i in range(0, 10, 2): print(i) # Ausgabe: 0, 2, 4, 6, 8

Vollständiges Beispiel

Hier ist ein vollständiges Beispiel einer For-Schleife in Python:

# Schleife von 0 bis 4 for i in range(5): print(f"Die aktuelle Zahl ist: {i}") print(f"Das Quadrat von {i} ist: {i*i}") print("Schleife beendet!")

Ausgabe:

Die aktuelle Zahl ist: 0 Das Quadrat von 0 ist: 0 Die aktuelle Zahl ist: 1 Das Quadrat von 1 ist: 1 Die aktuelle Zahl ist: 2 Das Quadrat von 2 ist: 4 Die aktuelle Zahl ist: 3 Das Quadrat von 3 ist: 9 Die aktuelle Zahl ist: 4 Das Quadrat von 4 ist: 16 Schleife beendet!
