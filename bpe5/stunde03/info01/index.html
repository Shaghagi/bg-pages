Was ist eine For-Schleife?

Eine For-Schleife ist eine Kontrollstruktur in der Programmierung, die es ermöglicht, einen Codeblock mehrmals auszuführen. Sie besteht aus drei Teilen:

  1. Initialisierung: Setzt die Schleifenvariable auf einen Startwert
  2. Bedingung: Überprüft, ob die Schleife weiterlaufen soll
  3. Inkrement: Verändert die Schleifenvariable nach jedem Durchlauf
